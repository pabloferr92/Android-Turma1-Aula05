📖 Sobre a MainActivity.kt

A MainActivity.kt é uma classe Kotlin que representa a primeira tela que o usuário vê ao abrir o app. Nela:

A classe herda de AppCompatActivity, permitindo recursos modernos de UI.

No método onCreate(), o layout XML é carregado usando setContentView(...).

É aqui que você começa a adicionar lógica para eventos e elementos visuais.

▶️ Como executar o projeto

Para rodar esta aplicação no Android Studio:

Clone o repositório:

git clone https://github.com/pabloferr92/Android-Turma1-Aula05.git

Abra o projeto no Android Studio

Aguarde as dependências carregarem

Conecte um dispositivo físico ou emulador

Clique em Run (▶️)
